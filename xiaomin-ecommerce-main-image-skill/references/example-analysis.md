# 主图样张分析输出

Use this file when the user provides one or more existing main images and asks to analyze principles, learn from them, improve the skill, or redesign based on them.

## Analyze Each Example By

For each image, identify:

1. Product/category.
2. Main image type.
3. Layout archetype.
4. Product scale and placement.
5. Dominant click reason.
6. Offer module, if any.
7. Typography hierarchy.
8. Color system.
9. What to borrow.
10. What to avoid.

## Common Patterns To Extract

Look for:

- top event strips
- bottom price/coupon bars
- membership/gift cards
- giant benefit headline
- product/texture close-up hero
- small SKU confirmation packshot
- fan-shaped SKU lineup
- rounded white content cards
- high-contrast price numerals
- platform event identity
- brand campaign mood fields

## Output Format

Use concise bullets:

```text
样张归纳：
- 图 1：活动价格型。结构：顶部活动条 + 中部产品利益 + 底部价格券。可借鉴：...
- 图 2：食欲质感 + 价格券型。结构：...

抽象规则：
1. ...
2. ...

对 skill 的更新建议：
- main-image-types: ...
- promo-layouts: ...
- prompt-template: ...
- compliance-check: ...
```

## Do Not

- Do not copy exact brand layouts as a fixed template.
- Do not treat all examples as generally appropriate for every platform.
- Do not infer prices, gifts, or activity dates for new products from examples.
- Do not claim a design is premium just because it has more information.
