# 小敏电商主图设计 Skill 精简使用手册

## 1. 用途

`xiaomin-ecommerce-main-image-skill` 用于设计、规划、生成和复盘电商主图。

它适合：

- 产品主图
- 活动主图
- 卖点主图
- SKU 组合主图
- 价格/券/会员机制主图
- 食品、宠物、日化、美妆、饮料等商品主图
- PSD-ready 分层交付包

核心原则：**先确认信息，再做图。**

## 2. 使用方式

在对话中输入：

```text
$xiaomin-ecommerce-main-image-skill 做一张产品主图
```

也可以直接给产品图和需求，例如：

```text
$xiaomin-ecommerce-main-image-skill
做一张德芙经典巧克力主图。
卖点：一口沉醉 乐享丝滑。
夏日 icon：冰袋冷运。
不展示价格。
```

## 3. 它会先确认什么

每次新主图需求，skill 会先整理信息确认卡：

```text
产品名称：
品牌：
类目：
规格/SKU：
目标平台：
主图类型：
价格信息：
活动/优惠机制：
赠品/会员机制：
活动时间：
核心利益点：
必须展示文案：
不展示/禁用信息：
素材质量：
缺失信息：
```

你确认后，它才会继续规划或生成主图。

## 4. 提需求建议

最好一次性提供：

```text
产品名称：
产品图：
卖点：
价格：
优惠机制：
赠品/会员权益：
物流/服务：
目标平台：
必须展示文案：
不能展示的信息：
```

没有的信息请明确写：

```text
价格：不展示
赠品：无
活动时间：不展示
```

## 5. 常见主图类型

- 白底搜索主图：突出商品本身
- 卖点主图：突出 1-3 个核心卖点
- 质地主图：突出口感、质地、材质、使用效果
- SKU 组合主图：突出多口味、多件装、多规格
- 活动主图：突出价格、券、满减、赠品、会员机制
- 服务机制主图：突出冷运、送货上门、入会权益等

## 6. 可选表现模板

主图可以指定表现模板：

- `产品质感型`：产品大图 + 真实质地，适合冻干、零食、罐头。
- `宠物场景型`：宠物 + 产品 + 食盆/使用场景，适合宠物食品和用品。
- `品牌信任型`：品牌区 + 大产品 + 稳重排版，适合高信任品牌。
- `服务机制型`：底部或侧边展示入会、顺丰、冷运等机制。
- `活动转化型`：价格、券、满减、赠品强展示。
- `白底搜索型`：干净货架图。

宠物类默认可在 `产品质感型`、`宠物场景型 + 品牌信任型`、`服务机制型` 中选择。服务机制必须由你提供，skill 不会自动编造。

## 7. 注意事项

不要让主图展示未确认的信息，包括：

- 价格
- 优惠券
- 满减
- 赠品
- 会员权益
- 活动时间
- 平台官方标识
- 认证/奖项
- 医疗功效
- 绝对化表述，如第一、最强、100%

如果没有高清产品图，生成结果只能作为方向稿，不能当正式包装稿。

## 8. 输出结果说明

- `可作为方向稿`：结构和调性可用，但还需要高清素材精修
- `接近正式稿`：商业完成度较高，可继续细修
- `不合格需重做`：有乱码、商品变形、乱加信息或画面不成立

## 9. PSD 分层输出

当你需要源文件时，可以这样说：

```text
这张主图做 PSD 分层文件
```

skill 会按优先级输出：

- `true PSD`：当前环境支持写 PSD 时，输出 `.psd`。
- `PSD-ready 分层包`：输出透明 PNG 图层、预览图、`layer-manifest.json` 和图层说明。
- `layer spec only`：仅做规划时，输出图层结构清单。

PSD 分层会按背景、产品、质感/道具、阴影、高光、文字、角标、优惠模块、安全区等层级拆分。  
如果原图本身是 AI 生成的扁平图，不能假装所有内容都可编辑；只能把可重建的文字、角标、优惠模块优先做成可编辑/可替换层。

## 10. 一句话记住

主图不是越热闹越好，而是要把**产品、价格/机制、利益点、禁用信息**先确认清楚，再决定画面怎么做。
